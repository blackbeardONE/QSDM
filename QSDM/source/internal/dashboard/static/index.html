Bounded ring buffer of the most recent rejection records plus per-field rune-truncation counters and the full persistence lifecycle (errors / compactions / on-disk record count). Truncation is applied before records reach the ring, so high truncated counts indicate possible hostile input — not data loss in the buffer itself.
