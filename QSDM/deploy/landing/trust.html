Attestation transparency

QSDM validators may optionally publish a signed NVIDIA NGC proof bundle as a transparency signal. The widget below shows how many validators on this network have published a fresh attestation within the last 15 minutes, over the total number of publicly-known validators.

Scope: NVIDIA-lock is an opt-in, per-operator API policy — not a consensus rule. Consensus runs on CPU via Proof-of-Entanglement + BFT and admits validators without a GPU. A missing or stale attestation never causes a validator to be excluded from consensus, block proposal, or finality. See NVIDIA_LOCK_CONSENSUS_SCOPE.md.
