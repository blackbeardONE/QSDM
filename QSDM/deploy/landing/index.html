Mining and edge participation

QSDM keeps NVIDIA protocol mining visible and premium, while still giving CPU-only users a path through shared edge tasks.

NVIDIA-attested protocol mining

QSDM Miner

The miner task requires an eligible NVIDIA Turing-or-newer GPU for v2 identity, attestation, and CUDA SHA3/DAG proof search. Hive fails closed if GPU solving is unavailable. The Tensor-Core fork is a separate future consensus activation. Advanced operators may use the console miner; there is no separate consumer GUI miner release.
