Wallet · Browser + CLI

Self-custody, in 30 seconds

Generate an ML-DSA-87 keypair, encrypt it with your passphrase, sign and send transactions — entirely in your browser. Same WASM code as the validator, no extension required.

Miner · NVIDIA-locked · v2

Mine Cell on your GPU

Mining mints CELL from attested NVIDIA proofs. Stake 10 CELL, register your (node_id, gpu_uuid), run qsdmminer-console. An entry-level RTX 3050 is already 4× a 32-thread Xeon at n=4096.

Download miner Quick-start
Validator · CPU-only · VPS

Run a validator

Validators run Proof-of-Entanglement + BFT on CPU-only VPS-class hardware and earn transaction fees. No GPU required; NGC attestation is an opt-in transparency signal, never a consensus gate.

Attestation transparency

Every NGC attestation (NVIDIA GPU Confidential Computing) and every PoE+BFT consensus round produces a receipt anyone can verify against the chain. Public attestation index at /api/v1/trust/attestations/*; same JSON the operator dashboard tile consumes.
