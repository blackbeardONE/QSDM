NVIDIA-only protocol mining

QSDM Miner

The miner task is for supported NVIDIA GPUs only. Minimum path: NVIDIA Turing or newer, CUDA compute capability 7.5+, working NVIDIA driver/nvidia-smi, and a funded QSDM signer wallet. Advanced operators may use the console miner; there is no separate consumer GUI miner release.
